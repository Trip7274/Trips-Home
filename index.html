What I do

I work on a few programming projects between school and stuff, usually Sofa API, but there are some unreleased projects.
Sofa is an open-source backend for hobbyist homeservers built using C#, which handles all the tasks of fetching server stats and metrics, software management, and such, as a REST API for other frontends to use.

This allows you to use (or write) anything from a powerful, full-featured server dashboard, to a simple, no-fuss stat display, without having to worry about exactly how to fetch and manage it server-side.
Got obscure hardware that doesn't work out of the box? Sofa fetches its stats through simple shell scripts, so you don't need to learn C# to contribute!
Though, Sofa is still in development, so please do submit issues or code to help out!
